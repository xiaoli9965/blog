      # 1. 🌈Redis 安装 (版本 5.+)

      # 1. 安装 GCC 环境

      由于 Redis 由 C 语言开发,安装 Redis 时需要有 Gcc 环境

      #需要以root权限执行以下命令 ,并且可以连接外网
yum -y  install gcc automake autoconf libtool make

      # 2. 安装 （单机）

      #进入 opt目录
cd /opt
# 通过wget命令下载Redis
wget http://download.redis.io/releases/redis-5.0.8.tar.gz
#解压
tar xzf redis-5.0.8.tar.gz
cd redis-5.0.8
#编译
make
# 安装 PREFIX={安装目录}
make PREFIX=/opt/redis5 install

      # 2.1. 启动

      # 进入Redis目录
cd /opt/redis5/bin/
# 启动服务端
./redis-server

      🔥 带配置的启动: /redis-server 配置文件路径

      看到如下消息就代表启动成功了

      查看 redis 进程

      # 查看redis进程
ps -ef|grep -i redis

      # 2.2. 测试

      上面步骤 redis 启动不是后台启动,我们无法输如 linux 命令.
      所有需要重新在开一个连接窗口进行操作

      # 进入redis目录
cd /opt/redis5/bin/
# 启动客户端
./redis-cli
# 存入一个值
set name xiaoli
# 取出一个值
get name

      # 2.3. 停止

      正常关闭:

      # 客户端执行命令 -会进行持久化
shutdown

      非正常关闭:可能造成数据丢失

      # 直接结束进程 没有实例化的数据会被丢失
kill -9 PID

      #创建redis-cluster目录
mkdir -p /opt/redis-cluster
#进入redis-cluster目录
cd redis-cluster/
#创建目录
mkdir 7000 7001 7002

      # 3. redis-cli 连接

      连接 redis:

      # 什么都不传的情况下默认连接 本机的:6379端口
./redis-cli -h ip地址 -p 端口

      退出客户端:

      • ctrl+c
      • 客户端内输 exit

      # 4. Redis 配置文件

      配置文件在路径 redis解压目录中(不是安装后的目录哦)
      路径:/opt/redis-5.0.8/redis.conf

      关键配置参数详解:

      字段 默认值 解释
      🔥bind bind 1270.0.1 绑定的 ip
      🔥port bind 6379 绑定的端口
      🔥daemonize daemonize no 是否后台启动 ,yes 等于后台启动
      protected-mode protected-mode yes 保护模式
      pidfile pidfile /var/run/redis_6379.pid 当 redis 以后台的方式启动时,会将线程信息写入这个文件
      databases databases 16 设置数据库数量,默认开启 16 个库 下标从 0 开始
      🔥save save 900 1 save 300 10 save 60 10000 设置 RDB 的触发时机, 默认900秒内1个更改,300秒内10个更改或 60秒内1W个更改
      rdbcompression rdbcompression yes RDB 时是否使用压缩算法,redis 采用的 LZF 算法(为了节省 CPU 压力可以关闭),关闭后 RDB 文件会变大
      dbfilename dbfilename dump.rdb 持久化文件名字
      🔥dir dir ./ 指定数据库存放目录
      requirepass 无 设置启动密码
      maxmemory 无 设置最大内存限制
      🔥appendonly appendonly yes 开启 AOF 模式

      # 4.1. 🔥 初始化 redis 时需要修改的配置

      • daemonize yes #开启后台运行
      • 注释掉 bind 1270.0.1 不注释的话外部无法连接 redis
      • requirepass (设置密码,需要的情况下.redis 每秒支持 1W 的 QPS,如果要设置的话需要设置非常强大的密码防止暴力破解)
