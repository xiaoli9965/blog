                                                                                                                                                                                                                                                            # Java 基础

                                                                                                                                                                                                                                                            # 1. JAVA 中的几种基本数据类型是什么, 各自占用多少字节.

                                                                                                                                                                                                                                                            • int 4
                                                                                                                                                                                                                                                            • long 8
                                                                                                                                                                                                                                                            • float 4
                                                                                                                                                                                                                                                            • duoble 8
                                                                                                                                                                                                                                                            • boolean 1bit
                                                                                                                                                                                                                                                            • char 2
                                                                                                                                                                                                                                                            • byte 1

                                                                                                                                                                                                                                                            # 2. String 类能被继承吗, 为什么.

                                                                                                                                                                                                                                                            • 不能,应为String 类被final修饰的,被fianl修饰的内不能被集成,binal修饰的变量不能修改值,final修饰的方法不能被重写

                                                                                                                                                                                                                                                            # 3. String, Stringbuffer, StringBuilder 的区别.

                                                                                                                                                                                                                                                            • String长度不可变,适用于少量的字符串操作的情况
                                                                                                                                                                                                                                                            • 以下两种适用于大量字符串操作的情况,比如for循环拼接字符串
                                                                                                                                                                                                                                                            • Stringbuffer 长度可变, 线程安全,部分方法通过synchronized来保证线程安全
                                                                                                                                                                                                                                                            • StringBuilder长度可变, 线程不安全

                                                                                                                                                                                                                                                            # 4. ArrayList 和 LinkedList 有什么区别.

                                                                                                                                                                                                                                                            • ArrayList底层是数组实现., 查找速度相对较快 o(1).
                                                                                                                                                                                                                                                              • 大数据量情况下Arraylist插入删除速度不如arraylist.
                                                                                                                                                                                                                                                            • LinkedList底层是链表实现, 链表需一个一个遍历所以相对速度慢一点 o(n).
                                                                                                                                                                                                                                                              • 但是插入速度比arraylist块,只需要新建一个node到相应位置连接起来就行了. 在做反转的时候性能优于ArrayList

                                                                                                                                                                                                                                                            # 5. 讲讲类的实例化顺序, 比如父类静态数据, 构造函数, 字段, 子类静态数据, 构造函数, 字段, 当 new 的时候, 他们的执行顺序.

                                                                                                                                                                                                                                                            • 父类--静态变量
                                                                                                                                                                                                                                                            • 父类--静态初始化块
                                                                                                                                                                                                                                                            • 子类--静态变量
                                                                                                                                                                                                                                                            • 子类--静态初始化块
                                                                                                                                                                                                                                                            • 父类--变量
                                                                                                                                                                                                                                                            • 父类--初始化块
                                                                                                                                                                                                                                                            • 父类--构造器
                                                                                                                                                                                                                                                            • 子类--变量
                                                                                                                                                                                                                                                            • 子类--初始化块
                                                                                                                                                                                                                                                            • 子类--构造器

                                                                                                                                                                                                                                                            顺序: 静态变量> 静态代码块> 变量> 代码块> 构造器

                                                                                                                                                                                                                                                            # 6. JAVA8 的 ConcurrentHashMap 为什么放弃了分段锁, 有什么问题吗, 如果你来设计, 你如何设计.

                                                                                                                                                                                                                                                            • 使用了cas机制,提升并发
                                                                                                                                                                                                                                                            • 链表+红黑树

                                                                                                                                                                                                                                                            # 7. 有没有有顺序的 Map 实现类, 如果有, 他们是怎么保证有序的.

                                                                                                                                                                                                                                                            • treeMap: key升序, 顾名思义二叉树结构
                                                                                                                                                                                                                                                            • LinkedHashMap: 插入顺序, 链表+hash表, 链表记录顺序

                                                                                                                                                                                                                                                            # 8. 抽象类和接口的区别, 类可以继承多个类么, 接口可以继承多个接口么,类可以实现多个接口么.

                                                                                                                                                                                                                                                            单继承(抽象类), 多实现(接口)

                                                                                                                                                                                                                                                            # 9. 继承和聚合的区别在哪.

                                                                                                                                                                                                                                                            • 继承就是获得父类的方法,直接调用
                                                                                                                                                                                                                                                            • 聚合就是将一个对象作赋值到变量中,然后获得其方法能力

                                                                                                                                                                                                                                                            # 10. IO 模型有哪些, 讲讲你理解的 nio , 他和 bio, aio 的区别是啥, 谈谈 reactor 模型.

                                                                                                                                                                                                                                                            • nio: 同步非阻塞
                                                                                                                                                                                                                                                            • bio: 同步阻塞
                                                                                                                                                                                                                                                            • aio: 异步非阻塞

                                                                                                                                                                                                                                                            # 11. 反射的原理, 反射创建类实例的三种方式是什么.

                                                                                                                                                                                                                                                            • 对象还为被jvm加载时:class.forName
                                                                                                                                                                                                                                                            • 已加载时可以使使用 类名.class
                                                                                                                                                                                                                                                            • 已经创建的实例:对象.getclass();

                                                                                                                                                                                                                                                            # 12. 反射中, Class.forName 和 ClassLoader 区别 .

                                                                                                                                                                                                                                                            这两个都会触发类加载

                                                                                                                                                                                                                                                            • ClassLoader 只负责将字节码文件加载到jvm虚拟机中, 后续由用户自己去new对象

public class Singleton01 {
    private Singleton01() {}
    private static volatile Singleton01 s = new Singleton01();
    public static Singleton01 getInstance() {return s;}
}

class Singleton02 {
    private Singleton02() {}
    private volatile static Singleton02 s = null;
    public static Singleton02 getInstance() {
        if (s == null) {
            synchronized (Singleton02.class) {
                if (s == null) {
                    s = new Singleton02();
                }
            }
        }
        return s;
    }
}

    }
