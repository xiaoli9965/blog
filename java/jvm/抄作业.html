      # 1. 为什么新生代不能用 CMS

      因为它的回收算法是Mark Sweep(标记清除)算法,而新生代需要的是Mark Copy标记复制算法(E 区拷贝到 S 区)

      # 2. 一个 Object 对象在 JVM 中占多少个字节

      Markword：8 字节 Class point：4 字节 panding：4 字节 Object o1 =new Object() 中的 o1 占 4 个字节

      所以一个 Obj 对象占 16+4 个字节
